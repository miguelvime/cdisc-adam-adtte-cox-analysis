Parámetros a tener en cuenta para hacer realistas los datos - En este ensayo publicado sobre AAA ocurre algún evento adverso en 80-90% de los participantes. (1)

  • En el mismo ensayo observamos los AE más reportados (infusion-related reaction, ARIA with microhemorrhages or hemosiderin deposits”, ARIA-E, Headache, Fall, Urinary tract infection, superficial siderosis of central nervous system). En los 3 primeros hay diferencia en la frecuencia de aparición a favor del grupo experimental, mientras que en los 2 últimos la diferencia es apenas imperceptible. Decidimos no escoger AE diferentes para los grupos porque parece más realista que se midan los mismos efectos adversos en ambos grupos y adaptar la frecuencia de aparición de acuerdo al ensayo clínico que uso para ver las proporciones. (1)

  • La gravedad de los AE dependerá de cuál sea, la cefalea y las caídas no tendrán un grado 4 según el CTCAE (Terminology Criteria for Adevrse Events). (2) Además los EA suelen ser más frecuentes cuanto más leves, por lo que ajustamos las probabilidades en ese sentido.

raw_ae<- raw_dm %>% sample_n(0.85*n) %>% 
  mutate(
    EVENTO = if_else(RAMA_ASIGNADA == "Anticuerpo monoclonal",
                     sample(c("reaccion infusional", "ARIA-H","ARIA-E","Cefalea","Caida"),
                            n(),
                            replace = TRUE,
                            prob = c(0.3,0.14,0.12,0.11,0.1)),
                     sample(c("reaccion infusional", "ARIA-H","ARIA-E","Cefalea","Caida"),
                            n(),
                            replace=TRUE,
                            prob = c(0.07,0.07,0.02,0.08,0.09))),
    GRAVEDAD = if_else(EVENTO == c("reaccion infusional","ARIA-H", "ARIA-E"),
                              sample(c(1,2,3,4),
                                     n(),
                                     replace=TRUE,
                                     prob = c(0.7,0.15,0.1,0.05)),
                              sample(c(1,2,3),
                                     n(),
                                     replace=TRUE,
                                     prob = c(0.7,0.2,0.1))),
    FECHA_INICIO_AE = FECHA_CONSENTIMIENTO + days(sample(5:40,
                                                         n(),
                                                         replace = TRUE)),
    FECHA_FIN_AE = FECHA_INICIO_AE + days(sample(2:15,
                                                      n(),
                                                      replace=TRUE))) %>% 
  select(ID_SUJETO,EVENTO,GRAVEDAD,FECHA_INICIO_AE,FECHA_FIN_AE) %>% 
  arrange(ID_SUJETO)

Compruebo que la fecha del fin del EA sea posterior al inicio para asegurar el realismo de los datos.

3. Estandarización a SDTM

Las variables siguen un código que facilita su seguimiento

Tabla DM- demographics

Algunas consideraciones:

  • El sexo en la terminología SDTM CT recoge los valores F para Female y M para Male. También están las posibilidades INTERSEX y U (unknown), sin sujetos en nuestro estudio. Exploramos cómo están en nuestras tablas para luego estandarizar los términos.

  • No tengo la fecha de nacimiento, por lo que escojo AGE antes que BRTHDTC, podríamos inferir el año de nacimiento con la edad y la fecha de consentimiento, pero ya estamos haciendo transformaciones en los datos.

unique(raw_dm$SEXO)

fechas_contacto <- raw_rs %>%

4.Derivación a ADAM

ADSL

ADSL contiene datos sobre demografía, exposición y disposición de un ensayo clínico.

Debemos consolidar edad,sexo, hospital y las variables de estratificación basal, en nuestro caso 3: presencia de APOE3, puntuación de ADAS-Cog y puntuación de Minimental. También añadimos las fechas temporales: consentimiento, fecha de progresión de la enfermedad y ultimo contacto.

ADSL <- DM %>%
  
  # 1. Inyección de Estratificación Gen APOE4
  derive_vars_merged(
    dataset_add = SC,
    by_vars = exprs(USUBJID),
    filter_add = SCTESTCD == "APOE4",
    new_vars = exprs(STRAT1 = SCSTRESC)
  ) %>%
  
  # 2. Inyección de ADAS-Cog Basal
  derive_vars_merged(
    dataset_add = QS,
    by_vars = exprs(USUBJID),
    filter_add = QSTESTCD == "ADASCGTL" & VISIT == "BASAL",
    new_vars = exprs(ADASBL = QSSTRESN)
  ) %>%
  
  # 3. Inyección de minimental (MSS1TN Basal)
  derive_vars_merged(
    dataset_add = FT,
    by_vars = exprs(USUBJID),
    filter_add = FTTESTCD == "MMS1TN" & VISIT == "BASAL",
    new_vars = exprs(MMSEBL = FTSTRESN)) %>% 
  
  # 4. Inyección de Tiempo
  derive_vars_dt(dtc = RFICDTC, new_vars_prefix = "RFIC") %>% 
  derive_vars_dt(dtc = RFPENDTC, new_vars_prefix = "RFPEN") %>% 
  derive_vars_merged(dataset_add = RS,
                     by_vars = exprs(USUBJID),
                     new_vars = exprs(RSDTC)) %>%
  
  # 5. Inyección de la variable deterioro sintomático
  
  derive_vars_merged(dataset_add = RS,
                     by_vars = exprs(USUBJID),
                     filter_add = RSTESTCD == "SYMPTDTR",
                     new_vars = exprs(SYMPTFL = RSSTRESC,
                                      PROGDT = as.Date(RSDTC))) %>%
  
  
  # 6. Derivaciones finales dependientes de las nuevas variables
  mutate(
    STRAT2 = case_when(
      MMSEBL >= 27 ~ "Normal",
      MMSEBL >= 12 & MMSEBL < 27 ~ "Sospecha patológica",
      MMSEBL < 12 ~ "Demencia"),
    
    TRT01A = ARM) %>%
  
  dplyr::select(STUDYID, USUBJID, SUBJID, SITEID, AGE, SEX, TRT01A, STRAT1,ADASBL,MMSEBL,STRAT2,SYMPTFL,RFICDT,PROGDT,RFPENDT)

ADTTE

ADTTE es la variable de ADAM que va a incluir el tiempo hasta el evento. Derivaremos tiempo hasta el evento (AVAL) en días y la variable de censura (CNSR: 0 = evento, 1 = Censurado).

ADTTE <- ADSL %>%
  mutate(
    # 1. Definición del Parámetro
    PARAMCD = "TTP",
    PARAM = "Tiempo hasta el Deterioro Sintomático (Días)",
    
    # 2. Fecha de inicio 
    STARTDT = RFICDT, 
    
    # 3. Fecha de Análisis (ADT): 
    ADT = case_when(
      SYMPTFL == "Y" ~ PROGDT,  # Fecha de progresión si ha progresado
      TRUE           ~ RFPENDT  # Fecha de último contacto si no progresó
    ),
    
    # 4. Censura (0 = Evento, 1 = Censura)
    CNSR = case_when(
      SYMPTFL == "Y" ~ 0,  
      TRUE           ~ 1   # Es de esta manera porque la censura depende del evento en nuestros casos
    ),
    
    # 5. Cálculo del tiempo (AVAL) y añado unidades (AVALU)
    AVAL = as.numeric(ADT - STARTDT+1),
    AVALU = "DAYS"
  ) %>%
  
  # 6. Selección final
  dplyr::select(STUDYID, USUBJID, TRT01A, STRAT1, STRAT2, 
                PARAMCD, PARAM, STARTDT, ADT, AVAL, AVALU, CNSR)

ADQS

ADQS <- QS %>%
  
  # 1. Filtro inicial y Fusión con la Demografía
  filter(QSTESTCD == "ADASCGTL") %>% 
  
  # 2. Cruce con ADSL usando TRT01A en lugar de ARM
  left_join(
    ADSL %>% dplyr::select(USUBJID, SITEID, AGE, SEX, TRT01A, STRAT1, STRAT2), 
    by = "USUBJID") %>%
  
  # 3. Variables ADAM
  mutate(
    PARAMCD = QSTESTCD,
    PARAM = "ADAS-Cog Total Score",
    AVAL = as.numeric(QSSTRESN),
    AVISIT = VISIT
  ) %>%
  
  # 4. Bandera Basal
  mutate(
    ABLFL = case_when(
      AVISIT == "BASAL" ~ "Y",
      TRUE ~ NA_character_
    )
  ) %>%
  
  # 5. Derivación de BASE, CHG y PCHG
  derive_var_base(
    by_vars = exprs(USUBJID, PARAMCD),
    source_var = AVAL,
    new_var = BASE
  ) %>%
  derive_var_chg() %>% 
  derive_var_pchg() %>%
  mutate(
    AVISITN = case_when(
      AVISIT == "BASAL" ~ 0,
      AVISIT == "MES 6" ~ 6,
      AVISIT == "MES 12" ~ 12)) %>%
  arrange(USUBJID, AVISITN) %>%
  relocate(AVISITN, .after = AVISIT) %>%
  
  # 6. Selección final 
  dplyr::select(STUDYID, USUBJID, SITEID, AGE, SEX, TRT01A, STRAT1, STRAT2, 
                PARAMCD, PARAM, AVISIT, AVISITN, AVAL, ABLFL, BASE, CHG, PCHG) %>%
  arrange(USUBJID, AVISITN)

ADFT

ADFT <- FT %>%
  
  # 1. Filtro inicial para el Minimental
  filter(FTTESTCD == "MMS1TN") %>% 
  
  # 2. Fusión con la Demografía (ADSL)
  
  left_join(
    ADSL %>% dplyr::select( USUBJID, SITEID, AGE, SEX, TRT01A, STRAT1, STRAT2), 
    by = "USUBJID"
  ) %>%
  
  # 3. Mapeo de variables estándar BDS
  mutate(
    PARAMCD = FTTESTCD,
    PARAM = "Mini-Mental State Examination Total Score",
    AVAL = as.numeric(FTSTRESN),
    AVISIT = VISIT,
    
    # Variable numérica para el orden cronológico
    AVISITN = case_when(
      AVISIT == "BASAL" ~ 0,
      AVISIT == "MES 6" ~ 6,
      AVISIT == "MES 12" ~ 12,
      TRUE ~ NA_real_
    )
  ) %>%
  
  # 4. Identificación de la Visita de Referencia (Baseline)
  mutate(
    ABLFL = case_when(
      AVISIT == "BASAL" ~ "Y",
      TRUE ~ NA_character_
    )
  ) %>%
  
  # 5. Cálculos de Eficacia (Copia del valor basal y cambios)
  derive_var_base(
    by_vars = exprs(USUBJID, PARAMCD),
    source_var = AVAL,
    new_var = BASE
  ) %>%
  derive_var_chg() %>% 
  derive_var_pchg() %>%
  
  # 6. Selección final y ordenación profesional
  dplyr::select(STUDYID, USUBJID, SITEID, AGE, SEX, TRT01A, STRAT1, STRAT2, 
                PARAMCD, PARAM, AVISIT, AVISITN, AVAL, ABLFL, BASE, CHG, PCHG) %>%
  arrange(USUBJID, AVISITN)

ADAE

ADAE <- AE %>%
  
  # 1. Fusión con la Demografía (ADSL)
  left_join(
    ADSL %>% dplyr::select(STUDYID, USUBJID, SITEID, AGE, SEX, TRT01A, RFICDT), 
    by = c("STUDYID", "USUBJID")
  ) %>%
  
  # 2. Renombrar a variables Adam
  mutate(
    
    TRTEMFL_PROXY_DATE = as.Date(RFICDT), 
    
    ASTDT = as.Date(AESTDTC),           # Analysis Start Date (Inicio del síntoma)
    AENDT = as.Date(AEENDTC)            # Analysis End Date (Fin del síntoma)
  ) %>%

  mutate(
    TRTEMFL = case_when(
      ASTDT >= TRTEMFL_PROXY_DATE ~ "Y",
      TRUE ~ NA_character_ 
    )
  ) %>%
  
  # 4. Cálculo del Día Relativo (ADY)
  
  mutate(
    ASTDY = as.numeric(ASTDT - TRTEMFL_PROXY_DATE) + 1,
    AENDY = as.numeric(AENDT - TRTEMFL_PROXY_DATE) + 1
  ) %>%
  
  # 5. Selección final OCCDS
  dplyr::select(STUDYID, USUBJID, SITEID, AGE, SEX, TRT01A, 
                AETERM, AEDECOD, AESEV, # Variables clínicas del evento
                ASTDT, AENDT, ASTDY, AENDY,    # Variables temporales de análisis
                TRTEMFL) %>%                
  
  # 6. Orden cronológico 
  arrange(USUBJID, ASTDT, AETERM)
